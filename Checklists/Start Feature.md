# Старт фичи

Используйте этот чек-лист при начале работы над новой фичей. Подробности в [[Operations/Start A Feature]].

## Создание рабочего пространства

- [ ] Тикет создан: `/aidd-new-ticket <TICKET>`
- [ ] Директория `docs/<TICKET>/` существует
- [ ] `.active_ticket` содержит правильный тикет
- [ ] `tasklist-<TICKET>.md` создан

## Заполнение идеи

- [ ] `<TICKET>.idea.md` заполнен (Problem, Business Goal, Scope, Non-goals, Dependencies, Acceptance Criteria)
- [ ] `<TICKET>.vision.md` заполнен

## Планирование

- [ ] Фазы определены в tasklist
- [ ] Полоса выбрана (Professional / Critical) -- см. [[Methodology/Lanes]]

## Ветка

- [ ] Ветка создана: `<TICKET>-описание`
