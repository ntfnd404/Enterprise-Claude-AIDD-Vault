# Чеклист первой сессии

## Цель

Убедиться, что рантайм Claude Code загружен и готов к повседневной работе.

## Предварительные требования

- Установлен Claude Code
- Проект открыт из корня репозитория
- В репозитории есть `.claude/settings.json`, `.mcp.json`, `CLAUDE.md`

## Команды

```bash
claude --version
claude
```

Внутри сессии:

```text
/config
/agents
/hooks
/mcp
```

## Ожидаемый результат

| Команда | Ожидание |
|---|---|
| `/config` | Показывает настройки проекта |
| `/agents` | Показывает `analyst`, `researcher`, `planner`, `implementer`, `reviewer`, `security-reviewer`, `qa` |
| `/hooks` | Показывает зарегистрированные хуки из `.claude/settings.json` |
| `/mcp` | Показывает настроенные MCP-серверы (зависит от стека) |

## Проверка событий хуков

`/hooks` должна показать следующие события (тир Standard/Enterprise):

| Событие | Режим | Назначение |
|---|---|---|
| `InstructionsLoaded` | async | Напоминание об активном воркфлоу и дисциплине полос |
| `SessionStart` (compact) | sync | Восстановление контекста после компактификации |
| `PreToolUse` (Write/Edit) | sync | Проверка гейта: блокировка правок без активного тикета |
| `PostToolUse` (Write/Edit) | sync | Автоформатирование исходных файлов (из технического адаптера) |
| `PostCompact` | sync | Повторная инъекция тикета/фазы/полосы/цели |
| `FileChanged` | sync | Ревалидация файлов воркфлоу |
| `ConfigChange` | sync | Валидация изменений настроек/навыков |
| `SubagentStart` | async | Информационный: правила делегирования |
| `SubagentStop` | async | Информационный: порядок гейтов |
| `TaskCreated` | async | Только командный режим |
| `TaskCompleted` | async | Только командный режим |
| `TeammateIdle` | async | Только командный режим |

## Инициализация нового проекта

Если в проекте ещё нет рантайма `.claude/`:

```text
/aidd-init
```

Создаёт `settings.json`, агентов, навыки, хуки, `CLAUDE.md`.

Смотрите [[Bootstrap New Project]] для полного руководства.

## Условие остановки

Остановитесь, если `/hooks` или `/mcp` пусты, или если агенты проекта не загрузились.
Запустите `/aidd-validate` для диагностики.
